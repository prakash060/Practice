## 🔑 Manual Configuration Required

You need to add these values to complete the setup:

### **Backend Environment Variables** (`.env` in pk-foodfactory-backend/)
```
# MongoDB
MONGODB_URI=mongodb://localhost:27017/pk-foodfactory

# Razorpay (Get from https://dashboard.razorpay.com/)
RAZORPAY_KEY_ID=rzp_test_your_key_id_here
RAZORPAY_KEY_SECRET=your_secret_key_here
RAZORPAY_WEBHOOK_SECRET=your_webhook_secret_here

# Server
PORT=5000
```

### **Frontend Environment Variables** (`.env` in pk-foodfactory/)
```
REACT_APP_API_URL=http://localhost:5000/api
```

### **Razorpay Dashboard Setup**
1. **Create Account**: Sign up at https://razorpay.com
2. **Get API Keys**: From Dashboard → Settings → API Keys
3. **Set Webhook**: 
   - URL: `https://yourdomain.com/api/payment/webhook`
   - Events: `payment.captured`, `payment.failed`
   - Copy webhook secret to `.env`

### **MongoDB Setup**
- Install MongoDB locally or use MongoDB Atlas
- Update `MONGODB_URI` with your connection string
